# SiK Radio Configuration scripts

Flashes radio firmware and configures default settings. Assumes radio is connected to /dev/ttyUSB0.

*RFD900+*

    ./setup_rfd900.sh

    
*3DR SiK*

    ./setup_3dr.sh
